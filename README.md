# Зимний Фестиваль Школьного Спорта 2026

Сайт фестиваля: [Демо](https://srgolubev.github.io/wssf/)

## Структура проекта

*   `index.html` — Основная страница.
*   `css/` — Стили.
    *   `input.css` — Исходный файл стилей (Tailwind + Custom).
    *   `style.css` — Скомпилированный файл (не редактировать вручную).
*   `js/` — Скрипты.
    *   `main.js` — Основная логика (меню, скролл-анимации).
    *   `snow.js` — Эффект снега.
*   `assets/` — Изображения, видео, шрифты.

## Разработка

Проект использует **Tailwind CSS**. Для внесения изменений в стили:

1.  Установите зависимости:
    ```bash
    npm install
    ```

2.  Запустите режим отслеживания изменений (Watch mode):
    ```bash
    npm run watch
    ```
    Теперь при изменении `index.html` или `js/*.js` файл `css/style.css` будет обновляться автоматически.

3.  Для однократной сборки (Build):
    ```bash
    npm run build
    ```

## Особенности реализации

*   **Гибридный скролл:**
    *   **Desktop:** Используется `<video>` для плавной анимации прокрутки (высокая производительность).
    *   **Mobile:** Используется `<canvas>` с последовательностью кадров (`.webp`), так как мобильные браузеры блокируют программное управление видео. Загрузка кадров происходит "лениво" (Lazy Loading) при появлении секции в области видимости.
*   **Доступность:**
    *   Анимации (снег, скролл-эффекты) отключаются, если в системе пользователя включена настройка "Уменьшение движения" (`prefers-reduced-motion`).
